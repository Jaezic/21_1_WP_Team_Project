자바역사
자바는 객체지향프로그래밍 언어로서 C/C++에 비해 간략하고 쉬우며 네트워크 기능의 구현이 용이하기 때문에 인터넷 환경에서 가장 활발히 사용되는 프로그래밍 언어입니다. 자바 프로그램은 운영체제의 종류에 관계없이 대부분의 시스템에서 실행 가능한 장점을 가지고 있기 때문에 널리 사용되는 프로그래밍 언어로 인기를 얻고 있습니다. 특히 웹 애플리케이션 개발 언어로 가장 많이 사용되고 있는 것이 자바 언어입니다. 자바는 네트워크를 연결하는 것이 용이하기 때문에 굉장히 다양한 분야에서 활용되고 있습니다. 자바를 활용하여 응용 프로그램을 개발할 수도 있고, 자바 기반인 JCP를 통하여 웹사이트를 개발 가능하며 안드로이드 애플리케이션도 자바 기반으로써 자바를 익히시면 개발 가능합니다.
